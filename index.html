Nível 2
Anatomia de um Prompt
As 6 partes que transformam uma pergunta genérica em um prompt profissional
1. Role
Defina um papel claro para o modelo adotar
2. Objetivo
Diga exatamente o que quer que ele faça — seja explícito
3. Contexto
Dê tudo que o modelo precisa: restrições, exemplos, dados
4. Raciocínio
Peça que pense passo a passo antes de responder
5. Formato de Saída
Especifique: lista, tabela, parágrafos, JSON
6. Condição de Parada
Defina limites: tamanho, escopo, quando parar
Exemplo completo
// Role
Aja como um consultor financeiro especializado em PMEs brasileiras.

// Objetivo
Analise o fluxo de caixa anexo e recomende 3 cortes de custo.

// Contexto
Empresa: varejo de roupas, 12 funcionários. R$180k/mês. Margem: 22%.

// Raciocínio
Explique a lógica de cada corte antes de recomendar.

// Formato
Tabela: Corte | Economia | Impacto

// Parada
Máximo 3 recomendações. Não sugira demissões.
Um bom prompt transforma uma IA genérica em um especialista do seu negócio
Nível 3
Os building blocks de uma automação
5 conceitos que você precisa conhecer para criar sua primeira automação
1. Gatilho (Trigger)
O evento que inicia a automação: novo email, formulário preenchido, horário agendado
"É o quando — quando isso acontecer, comece"
2. Ação (Action)
O que a automação executa: enviar email, criar registro, atualizar planilha
"É o faça — faça isso automaticamente"
3. API
Como sistemas diferentes se conectam. O cardápio de comandos que um software oferece a outros
"O garçom entre dois sistemas que não se conhecem"
4. Webhook
Uma URL que recebe dados em tempo real. Você dá o endereço e o sistema te avisa quando algo acontece
"Dar seu celular para receber uma notificação"
5. Credenciais
A chave que permite a automação acessar suas contas (Gmail, CRM, planilhas)
"A senha do Wi-Fi — sem ela, nada conecta"
Com esses 5 conceitos você já consegue montar sua primeira automação em qualquer ferramenta
