Nível 2
Anatomia de um Prompt
As 6 partes que transformam uma pergunta genérica em um prompt profissional
1. Role
Defina um papel claro para o modelo adotar
2. Objetivo
Diga exatamente o que quer que ele faça — seja explícito
3. Contexto
Dê tudo que o modelo precisa: restrições, exemplos, dados
4. Raciocínio
Peça que pense passo a passo antes de responder
5. Formato de Saída
Especifique: lista, tabela, parágrafos, JSON
6. Condição de Parada
Defina limites: tamanho, escopo, quando parar
Exemplo completo
// Role
Aja como um consultor financeiro especializado em PMEs brasileiras.

// Objetivo
Analise o fluxo de caixa anexo e recomende 3 cortes de custo.

// Contexto
Empresa: varejo de roupas, 12 funcionários. R$180k/mês. Margem: 22%.

// Raciocínio
Explique a lógica de cada corte antes de recomendar.

// Formato
Tabela: Corte | Economia | Impacto

// Parada
Máximo 3 recomendações. Não sugira demissões.
Um bom prompt transforma uma IA genérica em um especialista do seu negócio
